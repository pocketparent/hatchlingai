# Hatchling 🐣

A lightweight baby memory journal built with React + Vite + Flask.

## 🧰 Tech Stack

- Frontend: React, TailwindCSS, Vite
- Backend: Flask (Python)
- Auth: Twilio SMS magic link
- AI: OpenAI GPT-4 + Whisper
- Storage: Firebase (media upload)
- Deployment: Render
